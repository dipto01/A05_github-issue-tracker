Q&A:

1️⃣ What is the difference between var, let, and const?
Ans: 	
    both var and let can be reassigned with different values but,
	var is an older way of declaring variable that caused problem with scoping.
	let has blocked scoping so easier to handle
	And const is used when the value assigned to the function doesn't need to change

2️⃣ What is the spread operator (...)?
Ans:	
    A spread operator basically "spreads" or separates elements inside an array or object 
	so that we can directly use the values inside array or object

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:	
    map() = can directly modify the elements of an array and make a new one
	forEach() = can access all elements inside an array like a for of loop
	but, does not return anything by default
	filter() = basically filters out elements inside an array by the condition applied

4️⃣ What is an arrow function?
Ans:	
    basically a very short way to write functions using the arrow operator '=>'
	we can even write anonymous functions with it and also

5️⃣ What are template literals?
Ans:	
    It is a modified version of string that is written using backticks (` `)
	It allows using variables or expressions using ${varName} and supports multi-line strings
